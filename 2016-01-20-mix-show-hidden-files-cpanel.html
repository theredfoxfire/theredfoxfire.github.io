How to show hidden files in cpanel

By default, when you use the File Manager in cPanel, it does not show hidden files.
Hidden files are any file that begins with a dot, such as .htaccess.
.htaccess is actually a common file to edit, and so if you would like to edit the file using your File Manager, you'll need to ensure you have the option selected to show hidden files.

  1. Log in to your cPanel
  2. Under Files click File Manager
  3. Slick Settings in the upper right corner
  4. Select your Document Root and check the box for Show hidden files and click Save
